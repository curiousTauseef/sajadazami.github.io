About Me

I'm an undergraduate student in CEIT department of AUT majoring in Computer Engineering. I'm interested in applying Machine Learning methods on Computer Vision, Robotics and Bio-Inspired problems. I'm also interested in Computational Neuroscience and Cognitive Sciences on a broader horizon.

Currently, I'm working on Object-Aware Person Recognition of PIPA Dataset(Computer Vision), and Comparing Corresponding fMRI data with ECOG and EEG data in Human VTC in Royan Institue of Research collaborating with Prof. Khaligh-Razavi from MIT(Neuroscience, Deep Learning).

Recent Work

Object-Aware Person Recognition of PIPA Dataset

CRLab, August 2017-Present

In this research, we try to use biometric features, alongside with face recognition and pose-specific models to create an online person recognition system which can learn to recognize new classes over time. . More description will be added soon. This work is being done under supervision of Dr. Nickabadi.

Comparing Corresponding fMRI, ECOG and EEG data in Human VTC

Royan Institute of Research, August 2017-Present

This research is focused on validating the correlation between corresponding ECOG, EEG and fMRI data. The data are collected from epilepsy patients and we will design a task for showing same active regions in the brain by Blood-oxygen-level dependent and related signals. This research is supported by Royan institute under supervision of Dr. Khaligh-Razavi.

Realistic Simulation of Wireless Communication in Multi-Robot Scenarios

CRLab and AIRLab Joint Project, July 2017-Present

Development of a communication node that interacts both with the ROS nodes of the robots and with a simulator(Gazebo), in order to receive requests and uses various models of signal propagation to decide if two robots can communicate or not. We also evaluate this node's impact on tasks like SLAM or multi-agent exploration.

Humanoid Robot Body Detection Using Deep Learning

CRLab and Bio-Inspired Lab Joint Project, Spring 2017

We compare several two-stage detection systems based on various CNN's and highlight their speed-accuracy trade off. The approach performs edge based image segmentation in order to reduce the search space and then a CNN validates the detection in the second stage. Results were published on RoboCup Symposium 2017 and are available here. Code and instructions are available here.



Virtual Robot Rescue Simulation Base Code

CRLab, Spring 2016-Fall 2016

Base code of SOSVR team for Virtual Robot Competition, including behaviour, human detection, navigation and bring-up packages. Base code version 2 is released later in spring 2017. Codes for version 2 are available here: Github. Find description paper here: paper.

Parametric and Non-Parametric Time Series Analysis for Power Usage Data

SML Course, Fall 2016

In this project, our goal was to predict the power usage of Ontario state, Canada, using recorded usage data from January 2002 to December 2016. We implemented parametric and non-parametric approaches like ARIMA and GMM to Time Series Analysis and compared them with respect to various evaluation metrics. Results are available in this report. Codes are available here.

UCI Heart Disease Classification and Comparison Using Various Probabilistic Models

SML Course, Fall 2016

In this project, we implemented probabilistic models on a data-set of heart disease(UCI Heart Disease Data). Pre-processing operations like discretizing continuous values and filling missing values were done. 2 different PGM and 3 Bayesian Networks using different subsets of data were used and evaluated. Codes are available here.

Implementing and Analysing Linear Regression On a Data Set of Student Grades

SML Course, Fall 2016

In this project, We implemented Ridge and Lasso on 2 different data-sets. Various evaluation metrics like RSS, R2, AIC and LOOCV were calculated and analyzed, on both train and test sets. Backward and Forward methods of Model Selection were examined and results were compared. Codes are available here.

Face Recognition Using SVD

EM Course, Fall 2016

This project is an implementation of this paper, a real-time face recognition system. The approach is essentially to apply the concepts of vector space and subspace to face recognition

Credit Card Transaction Fraud Detection

Fanavard Data Mining Contest, Spring 2016

This was a project done for a data mining contest. The goal was to classify credit card transaction as fraud or non-fraud. Codes are available here.
