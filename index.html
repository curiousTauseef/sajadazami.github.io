Statistical Machine Learning - Fall 2016

Implementing and Evaluating Naive Bayes Model and PGM for Classification of Illness

Amirkabir University of Technology, Tehran, Iran

In this project, we implemented naive bayes model on a data set of heart disease.(UCI Heart Disease Data). Pre processing operations like discretizing continuous values and filling missing values were done. 2 probabilistic graphical model and 3 bayesian network using different subsets of data were designed and results are available in technical report here Report. Codes are available on Github.

Statistical Machine Learning - Fall 2016

Implementing and Analysing Linear Regression on a Data Set of Student Grades

Amirkabir University of Technology, Tehran, Iran

In this project, I implemented Simple Linear Regression and Lasso on 2 different datasets. Various evaluation metrics like RSS, R2, AIC and LOOCV were calculated and analyzed, on both train and test sets. Backward and Forward methods of model selection were examined and results were compared. Results are available in technical report here Report. Codes are available on Github.
